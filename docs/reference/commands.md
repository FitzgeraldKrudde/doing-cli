# Commands

All commands are documented in the CLI app:

```console

Usage: doing [OPTIONS] COMMAND [ARGS]...

Options:
  --help  Show this message and exit.

Commands:

  nothing  Take a break.
  open     Quickly open certain links
  workon   Work on a new work item.
```
